# Sweet Shop Backend

This backend is built with Node.js, TypeScript, Express, and TypeORM using SQLite for persistence.

Setup:
- Copy `.env.example` to `.env` and set JWT_SECRET
- npm install
- npm run dev

My AI Usage:
I used GitHub Copilot to scaffold several files and ChatGPT to design the test cases and route structure. I edited and reviewed all generated code manually.
